Flight Time Tracker

группа: ИС-О-22\1 
задание: 1 задание 
автор: Горбунов Максим
программа: Flight Time Tracker 
язык: Java SE

Описание Данная программа рассчитывает налет летного состава на основе данных о рейсах. На выходе формируется отчет в формате "JSON", где указано:
- Суммарное полетное время каждого специалиста за каждый месяц.
- Проверка на переработки: (Превышение 8 часов в день, превышение 36 часов в неделю, превышение 80 часов в месяц)
